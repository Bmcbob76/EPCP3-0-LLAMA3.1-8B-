## 🚨 TROUBLESHOOTING

### **MCP Servers not starting:**

### **Port 8343 in use:**
Change PORT in config.json

### **OCR not working:**
Windows API server needs Tesseract installed

### **VS Code not responding:**
VS Code extension needs to be running on port 9001
